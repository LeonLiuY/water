Usage

  • Add a new page
    1. Enter a relative url you like in this web domain
    2. If the url is not used, you can see the edit page(Change the url if it is used)
    3. Input the whole HTML content in the edit page
    4. Click "Save"
  • Modify a page
    1. Enter the url of target page with additional "?edit"
  • Delete a page
    1. Enter the url of target page with additional "?delete"
